# Todo Rs

A blazingly fast CLI todo app written in Rust with SQLite backend and due date tracking.

## Features

- cli
- productivity
- rust
- sqlite
- todo
- CLI

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Raphasha27/todo-rs.git
cd todo-rs
```
